# MaaBanGDream v0.3.0

第三个 MaaFramework 开发预览，扩展游戏内自动演出能力。

- 支持 1–99 轮自动演出。
- 歌曲模式仅保留当前曲目和每轮随机选曲，不提供按名称指定歌曲。
- 支持 Easy、Normal、Hard、Expert、Special 难度选择。
- 每轮结算后安全返回主页，再进入下一轮。
- 可从大多数普通非主页页面启动；未识别主页时每 1.5 秒发送 ESC，必要时处理登录、关闭和剧情节点。
- 修复第二轮开始按钮可能随机点击到按钮外的问题。
- 自动演出次数耗尽时仍立即停止，不点击开始按钮。

已由用户在真实 `1280×720 / DPI 240` 雷电模拟器上验收非主页启动恢复、Expert 难度和连续两轮流程。

本版本不包含 Profile、挑战演出、每日调度或实时演奏。
