# MaaBanGDream v0.2.0

第二个 MaaFramework 开发预览，新增当前歌曲单轮自动演出。

- 从主页进入自由演出，对游戏当前选中的歌曲执行一次自动演出。
- 只有确认游戏内自动演出已开启后才点击开始；开关最多尝试三次。
- 自动演出次数耗尽时停止任务，不点击开始按钮。
- 演出结果最长等待 300 秒，完成结算和剧情处理后返回主页。
- 恢复流程支持安全节点点击、任务停止检测和有上限的应用重启。
- 增加自动演出 Pipeline、模板来源哈希和恢复路径测试。

本版本不支持指定歌曲、多轮重演、Profile、挑战演出或实时演奏。

## 验收说明

- 已在 `1280×720 / DPI 240` 的雷电模拟器上分别验证自动演出关闭和已开启状态，两轮均完成结算并返回主页。
- 演出等待期间未触发 BACK；停止失败验收后未继续控制游戏。
- 本次未在真机构造自动演出次数耗尽状态；该路径仅完成模板来源哈希和 Pipeline 契约测试。
